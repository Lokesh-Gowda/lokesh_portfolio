About This Project

This is a Portfolio web page. Build from scratch using Angular.


Methods/Property Used in this project:

=> Generated the components, services, pipes

=> Using Bootstrap modal and forms

=> Install the JSON server to the project for storing the data

=> imported the Reactive form, FormControl, Formgroup, Formbuilder, validators, HttpClient, HttpClientModule, ApiService,

=> Used Binding( string interpolation, Property, Event, Two-way binding )

=> Used Directives( *ngIf, [ngClass], [ngStyle], *ngFor )

=> Used Life cycle hooks

=> Used Subscribe method to map the data

=> Used Router, routerLink to navigate the pages

Steps/procedure Used in this project:

Created a Modal and Form by using Bootstrap for the Feedback Form

Added the HttpClientModule in app.module.ts for communicating HttpClient

Generated the services using CLI - ng g s service_name to interact b/w Http methods

Created a Model for all data type

Then created object for posting data to json server

Then imported the ApiService

Then Subscribing our data which is maped via services

Created login and signup components using CLI

Created Routing for the project

Created register array of object for storing data

Fire API and methods for storing data in JSON file

Created Login system

Created Home,ABout,Contact,Course,Project pages using routerLink and added the neccessary details
